## REQUISITOS
- Nodejs:   Necessário para rodar o Vitejs

## INSTALAÇÃO: 

-Clone o repositório

-GitHub CLI
```
gh repo clone devbymarcos/jogo-da-memoria
```

-Git https
```
https://github.com/devbymarcos/jogo-da-memoria.git
```
-Execute

```
npm install
```

-Execute localmente com  desenvolvedor usando:

``` 
npm run dev

```
